WebGet.com Ruby Gem Install

First-time users: add our gem certificate and source. When you do this once, it works for all our gems. 

  sudo wget http://webget.com/webget.pem
  sudo gem cert --add webget.pem
  sudo gem sources --add http://webget.com

Install the gem with advanced options. 

  sudo gem install webget_ruby_secure_random --test --trust-policy HighSecurity

Notes

Do you have any questions, comments, suggestions, or feedback? Let us know, we’re happy to help. Our email is webget@webget.com

Do you want to create your own high security gems? Learn how at www.rubygems.org/read/chapter/21

To see your current gem certificate list: 

  sudo gem cert --list

Our cert looks like this: 

  /C=US/ST=California/L=San Francisco/O=WebGet.com/OU=WebGet.com/CN=WebGet.com/emailAddress=webget@webget.com
